Hello!

My name is Sam and im a (mostly) self taught software developer.
I've been interested in programming since secondary school where i studied computer science. During my GCSEs i used Python to write code, and learnt problem solving techniques such as pseudocode and flowcharts - both of which i still use to this day when creating complex projects.

I have a burning passion for programming as it is my hobby first and foremost and a good creative outlet for me. Now that im older with more experience with programming, im interested in entering the full stack development space - due to my interest and enjoyment using multiple languages in parallel to achieve a solution.

More about me

Throughout the years I've picked up and put down some technology, sometimes out of change of interest or for education. For example I used to create small games in unity back in 2019 during my summer holidays- but i changed interest after i started college to making more systems and web based applications. I also picked up java for a college unit of mine (where i created the Mentally Maths app), while i like working with Java i found that C# would work a lot better for what i was interested in doing. The most recent example of me putting down some tech has been my adoption of Linux (in the flavour of Ubuntu) as it helps me keep focused and avoid distraction for when i want to create something, or to work on something pre-existing.

As Programming is my hobby - when I’m not using VSCode to create something i like to research topics through the internet, or through books. Currently I’m 50 pages into Clean Code by Robert C. Martin, and i have the Pragmatic Programmer by Andrew Hunt & David Thomas to read next.

Outside of the Terminal, some of my hobbies include cooking, learning french, binging series on streaming sites and sometimes gaming.

I love using Git and GitHub to share code with whoever wants to see my work (please see either the portfolio page or the contact page).
